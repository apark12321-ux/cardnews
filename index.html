요즘돈 카드뉴스 자동 발행 센터

주제와 확인된 사실을 입력하면 카드 원고를 만들고, 발행 큐에 등록해 GitHub Actions가 정해진 시간에 자동 게시합니다.

자동화 흐름
작성 → 큐 등록 → 예약 실행 → PNG 렌더 → Cloudinary 업로드 → Instagram 게시

1. 카드 작성

발행 큐에 등록을 누르면 Vercel API가 GitHub posts/ 폴더에 JSON을 직접 생성합니다. 이후 GitHub Actions가 매일 09:00 KST에 queued 글을 하나씩 자동 게시합니다.
