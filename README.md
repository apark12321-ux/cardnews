# 요즘돈 카드뉴스 자동화 툴

인스타 카드뉴스 채널 **요즘돈** 운영을 위한 정적 웹앱입니다.

주제, 확인된 사실, 공식 출처, 운영자 관점 한 줄을 입력하면 다음 산출물을 자동 생성합니다.

- 13장 인스타 캐러셀 카드뉴스 원고
- 게시물 캡션
- 해시태그
- 발행 전 검수 체크리스트
- 캔바/미리캔버스에 붙여넣기 쉬운 Markdown 원고

## 실행 방법

별도 설치 없이 브라우저에서 실행합니다.

```bash
git clone https://github.com/apark12321-ux/cardnews.git
cd cardnews
```

그 다음 `index.html` 파일을 브라우저로 열면 됩니다.

## 파일 구조

```txt
cardnews/
├─ index.html
├─ styles.css
├─ app.js
├─ templates/
│  └─ workflow.md
└─ README.md
```

## 사용 순서

1. 이번 주 카드뉴스 주제를 정합니다.
2. 공식 출처에서 금액·기한·대상 조건을 확인합니다.
3. `index.html`을 열고 입력값을 채웁니다.
4. `카드뉴스 생성` 버튼을 누릅니다.
5. 생성된 Markdown 원고를 복사해 캔바/미리캔버스에 붙여넣습니다.
6. 발행 전 체크리스트를 검수한 뒤 업로드합니다.

## 운영 원칙

1. 금액·기한·소득기준은 발행 당일 공식 출처 기준으로 확인합니다.
2. “무조건”, “누구나”, “확정 지급” 같은 낚시성 표현을 쓰지 않습니다.
3. AI가 쓴 티가 나지 않도록 운영자 관점 한 줄을 반드시 넣습니다.
4. 카드뉴스 1장은 후킹, 마지막 장은 저장·공유·댓글·팔로우 중 하나의 행동 유도를 넣습니다.
5. 같은 원고를 캐러셀, 릴스, 스토리, 블로그 글로 재가공합니다.
