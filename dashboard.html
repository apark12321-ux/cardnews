완전 자동화 부품 점검

보안상 브라우저에서는 GitHub Secrets 값 자체를 확인할 수 없습니다. IG_USER_ID, INSTAGRAM_ACCESS_TOKEN 같은 비밀값은 Actions 실행 결과로만 간접 확인합니다.

이 화면의 역할

이 대시보드는 GitHub 공개 API를 읽어서 현재 저장소 상태를 보여줍니다. 별도 서버나 n8n 없이도 Vercel 정적 사이트에서 동작합니다. 자동 게시 자체는 GitHub Actions가 수행하고, 이 화면은 관제판 역할을 합니다.

요즘돈 자동화 대시보드 · GitHub API 기반 · 토큰/시크릿은 화면에 노출하지 않음
